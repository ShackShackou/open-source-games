# Terre réaliste + Corse 3D

Démo web interactive qui permet de :

- visualiser la Terre en 3D,
- zoomer avec la souris,
- lancer une transition d'atterrissage,
- explorer la Corse entière avec un relief 3D et plusieurs villes modélisées (bâtiments en volume).

## Lancer localement

```bash
python3 -m http.server 8000
```

Puis ouvrir `http://localhost:8000`.

## Contrôles

- **Souris gauche** : orbite autour de la scène
- **Molette** : zoom
- **Bouton "Atterrir en Corse"** : transition vers la vue détaillée de l'île
